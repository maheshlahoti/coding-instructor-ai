# Coding Instructor AI

A **dark-mode coding assistant** web app that helps you understand programming concepts, debug code, and learn best practices. Built with **HTML, CSS, JavaScript**, and powered by **Google Gemini AI API**.

---

## Features

- Dark mode dashboard with sidebar navigation
- Ask coding questions in **any language**
- Get **AI-generated detailed explanations** and code examples
- Displays stats like **Questions Solved**, **Languages Supported**
- Popular topics and tutorials
- Responsive UI for desktop and mobile
